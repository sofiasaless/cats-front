Do que os gatuxos se tratam

Este é um site desenvolvido com intuito inicial de exercitar habilidades Full-Stack, trata-se de um CRUD de gatinhos gatuxos, você poderá cadastrar um Gatuxo, visualizá-lo, editá-lo e deletá-lo. O site foi desenvolvido apenas com o básico de HTML, CSS e JavaScript, e para back-end foi usado Java, framework Spring Boot e banco de dados MySQL.
